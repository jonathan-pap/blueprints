# SVG community examples

Inspiration galleries — browse, don't reload.

- **Power BI Community SVG gallery** — https://community.fabric.microsoft.com/ (search "SVG measure")
- **Microsoft Doc** — Power BI conditional formatting via SVG ImageUrl
- **Kerry Kolosko's SVG library** — kerrykolosko.com (DAX SVG patterns)
- **DAX Patterns** — daxpatterns.com (some include SVG variants)
- **SQLBI articles** — search "SVG" for measure-based icon and bar patterns

## When to browse

- "Need a chart shape I don't have a template for" → start with Kerry Kolosko.
- "Need IBCS-style charts" → SQLBI articles.
- "Need a unique status indicator" → community gallery search.

## Save useful patterns

Drop new measures into `../../examples/<name>-measure.dax`, add a wrapper file under `../per-chart/<name>.md`, update `../_index.md`.
