# page/ — atomic files

- `add-page.md` — new page
- `rename-page.md` — preferred over add+delete (preserves bookmarks and refs)
- `delete-page.md` — remove a page
- `set-page-size.md` — change width/height
- `set-page-wallpaper.md` — page background image
- `add-page-title.md` — title textbox at (24, 24)
