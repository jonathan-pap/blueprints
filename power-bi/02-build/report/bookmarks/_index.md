# bookmarks/ — atomic files

- `create-bookmark.md` — snapshot a page state (selections, filters, visibility)
- `bookmark-navigator.md` — built-in visual that lists bookmarks for one-click switching

## When to use bookmarks

- Toggle between alternate views ("Trend view" vs "Detail view")
- Reset-button pattern (clear all filters)
- Page-level scenarios ("Q1 view", "Q2 view")
