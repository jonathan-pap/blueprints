# filters/ — atomic files

- `add-page-filter.md` — filter scoped to all visuals on a page
- `add-visual-filter.md` — filter on a single visual
- `configure-filter-pane.md` — show/hide the right-side pane and configure its content

## Rule

If you find yourself adding more than 3 slicers to a page, move the rest into the filter pane (`configure-filter-pane.md`).
