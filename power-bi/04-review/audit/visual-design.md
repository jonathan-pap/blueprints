# Visual-design checklist

Subjective dimensions Pbir audit can't catch. Walk through manually, open in Desktop for visual confirmation.

## Layout

- Equal spacing between visuals (horizontal AND vertical)?
- Equal page margins?
- Visuals aligned to a grid?
- 3-30-300 detail gradient (summary top-left, detail bottom-right)?
- Page has a title?
- No more than 4–6 KPIs at top, no more than 3 slicers (filter pane for extras)?

## Color

- Conditional formatting used sparingly?
- Colors muted and passive (not loud / bright)?
- Negative sentiment colors (red, orange) ONLY on negative values?
- Color paired with secondary cue (arrow, icon) — never color alone?

## Readability

- Font sizes ≥ 12pt for all visible elements?
- 14pt+ for chart labels?
- 16–24pt for titles?
- Limited font variety (Segoe UI / Segoe UI Semibold)?

## Information : ink

- Non-essential elements minimized? (lighter gridlines, hidden unnecessary axes/labels)
- Subtitles hidden when redundant with title?

## Accessibility

- Alt text on charts and images?
- Minimal drop shadows (vestibular issues)?
- No reliance on color alone for meaning?

## Sorting

- Visuals sorted descending by primary measure (unless time-based or ordinal)?

## Helpful elements

- Refresh date shown?
- Last-update timestamp visible?
- Links / context where relevant?

## Output
